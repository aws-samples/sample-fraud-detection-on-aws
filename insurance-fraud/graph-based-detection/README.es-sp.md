# Detección de Fraude Basada en Grafos

Este directorio contiene proyectos que utilizan bases de datos de grafos y redes neuronales de grafos para detectar fraude analizando las relaciones entre entidades.

## Proyectos

| Proyecto | Descripción |
|---|---|
| [Detección de Fraude en Seguros de Auto](auto-insurance-fraud-detection/) | Detecta fraude en seguros de auto usando Amazon Neptune ML para identificar anillos de colisión, talleres sospechosos y testigos profesionales en un grafo de 2000 reclamos de ejemplo |
