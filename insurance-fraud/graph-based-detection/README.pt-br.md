# Detecção de Fraude Baseada em Grafos

Este diretório contém projetos que utilizam bancos de dados de grafos e redes neurais de grafos para detectar fraude analisando os relacionamentos entre entidades.

## Projetos

| Projeto | Descrição |
|---|---|
| [Detecção de Fraude em Seguros de Auto](auto-insurance-fraud-detection/) | Detecta fraude em seguros de auto usando Amazon Neptune ML para identificar anéis de colisão, oficinas suspeitas e testemunhas profissionais em um grafo de 2000 sinistros de exemplo |
