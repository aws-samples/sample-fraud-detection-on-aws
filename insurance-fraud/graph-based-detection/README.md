This directory contains projects that use graph databases and graph neural networks to detect fraud by analyzing relationships between entities.

## Projects

| Project | Description |
|---|---|
| [Auto Insurance Fraud Detection](auto-insurance-fraud-detection/) | Detects auto insurance fraud using Amazon Neptune ML to identify collision rings, suspicious repair shops, and professional witnesses across a graph of 2000 sample claims |
